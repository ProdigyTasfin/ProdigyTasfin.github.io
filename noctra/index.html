Accessibility Service use

Noctra uses Android’s Accessibility Service to provide real-time app blocking, app usage limits, Anti-Doomscroll interventions and Sentinel Mode.

When the service is enabled, Noctra detects when an app selected for blocking or limiting is opened. It can then display the blocking screen, usage-limit warning or intervention configured by the user.

Accessibility access is required for these app-control features to work in real time. Noctra does not use this service to read passwords, payment information, private messages or text you type. Accessibility data is not sold or used for advertising.

The service is enabled manually through Android Settings and can be disabled at any time.
